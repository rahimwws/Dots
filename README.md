# 3dots

A React Native application built with Expo.

[GitHub Repository](https://github.com/rahimwws/Dots)

## Setup

```bash
npm install
```

## Development

```bash
npm start
```

## Run on Platforms

```bash
npm run android
npm run ios
npm run web
```

## Other Commands

- `npm run dev` - Start with cache cleared
- `npm run lint` - Run linter
- `npm run prebuild` - Generate native projects
